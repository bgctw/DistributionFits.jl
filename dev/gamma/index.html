Gamma distribution

The Gamma distribution has a scale and a shape parameter. It can be fitted using two quantiles.

d = fit(Gamma, @qp_m(0.5), @qp_uu(5));
median(d) ≈ 0.5 && quantile(d, 0.975) ≈ 5

Fitting by mean or mode is not yet implemented.
